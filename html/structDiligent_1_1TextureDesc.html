bool Diligent::TextureDesc::operator==(const TextureDesc& RHS) const

Tests if two structures are equivalent.

Parameters
RHS in - reference to the structure to perform comparison with
Returns
  • True if all members of the two structures except for the Name are equal.
  • False otherwise. The operator ignores DeviceObjectAttribs::Name field as it does not affect the texture description state.
