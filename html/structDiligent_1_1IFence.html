Diligent::IFence struct
#include <Graphics/GraphicsEngine/interface/Fence.h>

Fence interface.

Defines the methods to manipulate a fence object

Remark

When a fence that was previously signaled by IDeviceContext::SignalFence() is destroyed, it may block the GPU until all prior commands have completed execution.
